# Warehouse Service (Ruby on Rails)

Учебный проект по домену `warehousing`: управление складами, товарами и движением остатков.

## Реализованный функционал

- Регистрация и аутентификация пользователей через Devise.
- Авторизация через Pundit с ролями:
  - `worker` — только просмотр.
  - `manager` — просмотр + управление складами/товарами/операциями.
  - `admin` — полный доступ + управление категориями.
- Доменные сущности и связи:
  - `Warehouse`, `Category`, `Product`, `Inventory`, `StockMovement`, `User`.
- CRUD-интерфейс (ERB views):
  - Склады, товары, категории, складские операции.
- Бизнес-логика:
  - Приход/расход (`inbound`/`outbound`) с автоматическим пересчётом остатков.
  - Валидация невозможности расхода сверх остатка.
- Пагинация списков с Pagy.
- Unit и integration тесты (Minitest).

## User Stories (MVP)

- Как менеджер склада, я хочу создать склад и добавить товары, чтобы вести учёт в системе.
- Как менеджер склада, я хочу проводить операции прихода и расхода, чтобы остатки обновлялись автоматически.
- Как работник склада, я хочу видеть список товаров и остатков, чтобы понимать текущую доступность.
- Как администратор, я хочу управлять справочниками (категории), чтобы поддерживать чистую структуру данных.

